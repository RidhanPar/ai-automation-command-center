# Portfolio Copy

## Project title

AI Automation Command Center

## Short portfolio description

Built a Streamlit dashboard for collecting, scoring, and tracking internal automation ideas across teams. The project helps prioritize automation opportunities using impact, effort, confidence, and estimated time-saving value.

## Detailed portfolio description

AI Automation Command Center is a practical operations dashboard designed to manage workflow automation ideas from different business areas such as Support Operations, Workforce Operations, People Operations, Finance Operations, and Engineering Operations.

The dashboard includes automation roadmap tracking, priority scoring, KPI cards, estimated monthly time-saving calculations, and an idea intake demo. It shows how automation opportunities can be moved from scattered ideas into a structured roadmap that supports leadership visibility and data-driven decision-making.

## Resume bullets

- Built an AI Automation Command Center to collect, score, and track workflow automation ideas across internal business functions.
- Created priority scoring logic using impact, effort, confidence, and estimated monthly time-saving values to support automation roadmap planning.
- Developed a Streamlit dashboard with KPI cards, roadmap status tracking, priority ranking, and operational time-saving visualizations.
- Added a safe n8n-style workflow concept for employee request routing and approval prioritization.
