# GitHub Setup Notes

Use these steps from the project folder.

## 1. Check the app locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 2. Create a new GitHub repository

Suggested repository name:

```text
ai-automation-command-center
```

## 3. Initialize Git

```bash
git init
git add .
git commit -m "Initial AI automation command center project"
```

## 4. Connect GitHub remote

Replace the URL with your own repository URL.

```bash
git remote add origin https://github.com/YOUR_USERNAME/ai-automation-command-center.git
git branch -M main
git push -u origin main
```

## 5. Suggested GitHub repository description

```text
Streamlit dashboard for scoring, prioritizing, and tracking AI and workflow automation opportunities across internal business teams.
```
